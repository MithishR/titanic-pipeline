# Titanic EDA and Data Pipeline

Exploratory data analysis and data cleaning pipeline for the [Titanic dataset](https://raw.githubusercontent.com/datasciencedojo/datasets/master/titanic.csv). 

## Pipeline Overview

1. **Data Cleanup & Preparation**
2. **Exploratory Data Analysis**
3. **Feature Engineering & Export**


## Usage

Open `titanic.ipynb` and run all cells. The cleaned dataset will be saved as `titanic_clean.csv` in the project directory.
